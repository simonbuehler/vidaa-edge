VidaaEdge - Getting Started

Install custom apps on your VidaaOS-based TV.

Overview

This project provides a method to install custom pwas on your VidaaOS-based TV. Before you start, ensure that the old method is no longer supported by your TV. Please note that this project is still under development and may not work on all devices. Use it at your own risk. Credits go to BananaMafia for discovering the exploit. This blog post provided the inspiration for creating this project.


Warning

This project is not officially supported by VidaaOS or any of its affiliates. Use at your own risk.

This method theoretically allows to install apps, code, scripts, or files on your TV. Only use trusted hosts to avoid potential risks (the safest option is to self-host!).

The Hisense API provides several powerful functions, including:
-> Hisense_installApp
-> Hisense_GetNetworkInfo
-> Hisense_GetIPAddress
-> Hisense_FileRead
-> Hisense_FileWrite
and many more.

Projects like this one (or similar ones?) can theoretically cause harm if these functions are misused, especially through malicious apps or scripts. These risks can be amplified if JavaScript global functions are exploited. Always be aware of the potential dangers and act responsibly when using this method. For additional information read the following blog post for more information.

Details

The global functions can only be executed if the domain name is "vidaahub.com". For this reason, it is important to "fake" the domain name. This is achieved using a DNS server, which resolves domain names to IP addresses.

If you choose the self-hosted solution, make sure to configure the network settings of your VidaaOS device to use your self-hosted DNS server. This will ensure that the project is recognized as "vidaahub.com".

Alternatively, if you opt for an already-hosted version of this project, you will need to set the provided DNS server in your network settings. This will make the hosted project accessible as "vidaahub.com".

Get Available Hisense Functions

Discovers all available Hisense functions in the global scope. This can be used to dynamically retrieve a list of functions exposed by the VidaaOS platform. Refer to @global.d.ts for more details. Additionally, Iâ€™ve included the result of all available 'Hisense' functions from my TV. See also vidaa.service.ts for additional informations.

Example:

const availableFunctions = getAvailableHisenseFunctions();
console.log(availableFunctions); // Output: ["Hisense_GetDeviceID", "Hisense_GetFirmWareVersion", ...]

Get Firmware Version

Retrieves the firmware version of the device. Use this function to verify if the device supports VidaaOS global functions I guess.

Example:

const firmwareVersion = Hisense_GetFirmWareVersion();
console.log(firmwareVersion); // Output: "V00.01.00a.F0116"

Install App

Installs an application on the device using the `Hisense_installApp` API.

Example:

Hisense_installApp("123", "MyApp", "https://example.com/thumbnail.png",
"https://example.com/iconSmall.png", "https://example.com/iconBig.png",
"https://example.com/app", "store", (res) => {
  if (res === 0) {
    console.log("App installed successfully!");
  } else {
    console.error("Failed to install app.");
  }
});

Uninstall App

Uninstalls an application from the device using the `Hisense_uninstallApp` API.

Example:

Hisense_uninstallApp("123", (status) => {
  if (status) {
    console.log("App uninstalled successfully!");
  } else {
    console.error("Failed to uninstall app.");
  }
});

For additional support or troubleshooting, refer to the Github repository.
